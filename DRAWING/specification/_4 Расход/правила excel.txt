СВТУ :
преобразователь сам двухканальный - называем по ветке первого канала с номером 70# (пример: NDK10CF701)
пишем это в графе комплект для двух точек измерения по первой точке измерения, т.е.
для двух точек описанных ниже, комплект будет NDK10CF701 - для обоих


каналы назвыаем по ветками трубопровода с нормальным номером 00#
1-ый канал - NDK10CF001
2-ой канал - NDA20CF001
расходомерное устройство на каждый канал называем по каналу датчика расхода ...CF001QP01
на каждое расходомерное устройство ставятся 2-4 датчика давления
заказываются 2 или 4 (DN>=200, см. РУ) но подключаются только 2
даем им позиции приписывая A, B, C, D
NDK10CF001A
NDK10CF001B 
...
 вграфе cpu указываем KKS позицию вычислителя 7$$

к каждому каналу есть корректировка по температуре - датчик
его называем как обычный датчик темппературы по порядку на трубопроводе
NDK10CT### - очередным номером


EXCEL

4 в графе комплект - пишем KKS _преобразователя_ см файл СВТУ задание пример
