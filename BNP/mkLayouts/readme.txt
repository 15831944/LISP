файл mklayouts.LSP
Пояснения к программе :
Программа создает вкладки листов (Layouts) 
на каждый, отмеченный в модели, блок "format" или "format-spec" из папки \\srv-energo\acad\!_Стандарт\Blocks\

прогамма так же сразу настраивает параметры печати для каждого листа
с использованием принтера bnp_pdfgostplotter.pc3 из папки \\srv-energo\acad\!_Стандарт\PDF\Plotters\

ЗАПУСК программы производится из командной строки автокада путем ввода букв:
mklayouts[enter] (от регистра не зависит)
